<task type="checkpoint:human-action" gate="blocking">
  <name>Task 1: Add CARGO_REGISTRY_TOKEN to GitHub repository secrets</name>
  <action>
The user needs to add their crates.io API token as a GitHub repository secret. This cannot be automated — GitHub secrets must be added through the web UI or gh CLI by a human with admin access.
  </action>
  <instructions>
1. Get your crates.io API token (same one stored in `~/.cargo/credentials.toml` from Phase 19)
   - If you need to find it: `cat ~/.cargo/credentials.toml` — copy the token value
   - If you need a new one: visit https://crates.io/settings/tokens and create a token with `publish-new` and `publish-update` scopes
2. Go to https://github.com/pgdad/durable-rust/settings/secrets/actions
3. Click "New repository secret"
4. Name: `CARGO_REGISTRY_TOKEN`
5. Value: paste your crates.io API token
6. Click "Add secret"
  </instructions>
  <resume-signal>Type "done" after adding the secret, or describe any issues</resume-signal>
</task>
