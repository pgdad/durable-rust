**Step 1: Create scripts/deploy-ecr.sh**

Create a bash script that performs the targeted apply of ECR and IAM resources only. This script is the repeatable command for Phase 11's first deploy step.

```bash
#!/usr/bin/env bash
set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
REPO_ROOT="$(cd "$SCRIPT_DIR/.." && pwd)"
TF_DIR="$REPO_ROOT/infra"
PARALLELISM=5

echo "=== Deploying ECR + IAM (targeted apply) ==="
echo "Using: terraform -chdir=$TF_DIR"

# Verify prerequisites
"$SCRIPT_DIR/verify-prerequisites.sh" || { echo "Prerequisites check failed"; exit 1; }

# Init (idempotent)
terraform -chdir="$TF_DIR" init -input=false

# Targeted apply: ECR repo + lifecycle policy + IAM role + policies + random_id
terraform -chdir="$TF_DIR" apply \
  -target=aws_ecr_repository.examples \
  -target=aws_ecr_lifecycle_policy.examples \
  -target=aws_iam_role.lambda_exec \
  -target=aws_iam_role_policy_attachment.durable_exec \
  -target=aws_iam_role_policy.invoke_permission \
  -parallelism=$PARALLELISM \
  -auto-approve

# Output ECR URL for build pipeline
echo ""
echo "=== ECR Repository URL ==="
terraform -chdir="$TF_DIR" output -raw ecr_repo_url
echo ""
echo ""
echo "=== Resource Suffix ==="
terraform -chdir="$TF_DIR" output -raw suffix
echo ""
```

Make the script executable: `chmod +x scripts/deploy-ecr.sh`

**Step 2: Run the targeted apply**

Execute `scripts/deploy-ecr.sh` to create the ECR repository, lifecycle policy, IAM role, and IAM policy attachments. This will also create the random_id suffix (Terraform auto-includes dependencies).

Note: This step requires valid ADFS credentials. If credentials are expired, the script will fail at the prerequisites check.
