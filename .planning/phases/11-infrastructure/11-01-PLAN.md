  - `package_type = "Image"`
  - `image_uri = "${aws_ecr_repository.examples.repository_url}:${each.key}"`
  - `publish = true` (CRITICAL: required for real version numbers in aliases)
  - `timeout = 900`, `memory_size = 256`
  - `durable_config` block: `execution_timeout = 3600`, `retention_period = 7` (CRITICAL: field name is `retention_period` NOT `retention_period_in_days`)
  - `tags = { Style = each.value.style }`
- `aws_lambda_alias "live"` with `for_each = local.handlers`:
  - `name = "live"`
  - `function_name = aws_lambda_function.examples[each.key].function_name`
  - `function_version = aws_lambda_function.examples[each.key].version`
